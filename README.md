# AI Prompt Architect 🚀

A premium AI prompt engineering tool that transforms raw requirements into high-fidelity, fine-tuned prompts using a structured framework (Role, Instructions, Context, Example, Parameters, Output, Tone).

![AI Prompt Generator](https://raw.githubusercontent.com/vikaskpawar/AI-Prompt-Generation-UI/main/screenshot.png) *(Note: Placeholder for actual screenshot)*

## ✨ Features
- **Structured Framework**: Automatically generates prompts using the 7-component architecture.
- **Dynamic Personas**: Adapts the AI persona based on the requirement type (e.g., Code Expert, Strategic Consultant).
- **Premium UI**: Modern dark-mode interface with glassmorphism and smooth animations.
- **Copy & Download**: Export your generated prompts instantly as Markdown.

## 🛠️ tech Stack
- **React**: Frontend framework.
- **Vite**: Ultra-fast build tool.
- **Framer Motion**: Smooth interactive animations.
- **Lucide React**: Beautiful iconography.
- **Vanilla CSS**: Custom premium styling.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/vikaskpawar/AI-Prompt-Generation-UI.git
    cd AI-Prompt-Generation-UI
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open in browser**:
    Navigate to `http://localhost:5173` to start architecting prompts!
